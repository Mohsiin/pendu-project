# Pendu Project
Principe du jeu est : 

·         Une fois arrivée sur la page d'acceil, je dois saisir mon mot deviné

·         L'application génère un mot au hazard dont les lettres sont masquées

·         Je saisi et je clique sur valider ce qui a pour effet ou non de l'afficher dans le mot

·         une popup s'affche pour me félicite ou non, en fin de partie

## Getting Started

Pour avoir le projet sur votre local, il suffit de faire un git clone de https://github.com/Mohsiin/pendu-project.git ou de le télécharger

Après, il faut ouvrir la page html helloWorld.html dans votre navigateur

Ainsi vous pouvez commencer à jouer
